# BAZOOKA-Rhythm-Game-Skins
A python script that fetches the content from the unity game and can be customized.

# Installation
- Python >= 3.11
- ```pip install pillow unitypy tkinter colorama```

# Preview
[![Youtube Video]()](https://www.youtube.com/watch?v=8ozvxOS2qvE)

# Use
- Run the script once with ```py ChangeFiles.py```, which is going to create the ```Replacements``` folder.
- After the folder is created, just put any ```.png, .jpeg, .jpg``` file with the name from the list. Ex: ```Arrow.png``` which is the arrows.
- Run the script again, and done!
